## 関数の特徴
### 変数型の拡張
prototypeを利用すると、関数に関数（メソッド）を追加することができるようになる。  
しかし、詳細についてはここでは話さにようにする。

### 再帰
JavaScriptでももちろん再帰関数を書くことは可能。
```js
function fib(n) {
  return n < 2 ? n : fib(n - 2) + fib(n - 1);
}

fib(10);
```
